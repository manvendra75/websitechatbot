Streamlit Chat Widget Demo

This page demonstrates how to embed your Streamlit chatbot as an overlay widget on any website.

Features:

  • Floating chat button in bottom-right corner
  • Smooth open/close animations
  • Minimize functionality
  • Mobile responsive
  • Professional design

Integration Code:

<!-- Add this to your website -->
<script src="https://your-domain.com/chat-widget.js"></script>

Click the chat button in the bottom-right corner to see it in action!
